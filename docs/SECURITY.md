# Security Policy

## Supported Versions

Currently, only the latest version of CinePulse is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < Latest| :x:                |

## Reporting a Vulnerability

We take the security of CinePulse seriously. If you believe you have found a security vulnerability, please report it to us responsibly.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please send an email to [INSERT SECURITY EMAIL]. 

Include the following information in your report:
- Type of issue (e.g., XSS, SQLi, etc.)
- Step-by-step instructions to reproduce the issue.
- Any potential impact of the vulnerability.

We will acknowledge your report within 48 hours and provide a timeline for a fix if applicable.
