# CinePulse

A personal movie and series tracker built as a single-page app using vanilla JavaScript and TailwindCSS.

I built this to manage everything I watch in one place — movies, TV shows, anime, K-drama, and Turkish series — without using external tracking platforms.

It uses the TMDB API for discovery and recommendations, and localStorage to persist your data.
